SERVER OVERVIEW

Advanced Mining System

既存の枠組みに捉われない独自の採掘システム。ブロックを叩いた時にHPバーが表示され、ダメージを与えて削り切るような、これまでのマイクラにはなかった新しい採掘感覚を楽しめます。

Multi-Layered Elements

採掘以外にも広がる多彩な遊び。本格的な釣りや、自分の見た目を飾るコスメティック機能など、その日の気分に合わせて自由に過ごせる要素をたくさん用意しました。

Exclusive Architecture

他の採掘サーバーには存在しない、独自の採掘システムにより実現した機能を多数搭載。ツルハシの強化値や採掘スピード、オリジナルのバフ効果など、他では体験できない自由なカスタマイズが可能です。

TERMS OF SERVICE

【禁止事項】

  • ゲームを有利にする改造クライアント・ツール・テクスチャの使用
  • 差別的な発言や、他人を不快にさせる発言
  • 血や暴力表現などを含む、不適切なスキンの使用
  • サーバーや運営スタッフを強く侮辱・軽視する行為
  • バグを利用した不正行為や、それを広める行為
  • Ban回避の試みや、DDoSなどサーバーに負荷や被害を与える行為
  • 個人情報の公開、なりすましなど、他人やコミュニティに迷惑をかける行為
  • パックデータやマップデータを不正に取り出し、使用・配布する行為
  • 売名目的の行為や、他サーバーへの勧誘・引き抜き行為
  • 他のプレイヤーのプレイをわざと邪魔する行為
  • 運営が想定していない動作や仕様を利用して有利になる行為

【非推奨事項】

  • UIや文字表示に影響を与えるテクスチャの使用
  • 非常に長時間にわたる連続プレイ
  • 回線が不安定な状態での接続

※これらは直ちに処罰されるものではありませんが、状況によっては制限を行う場合があります。

【許容事項】

  • サーバーに関する公開情報の共有
  • 常識の範囲内での配信や動画投稿
  • 見た目のみが変わる軽いテクスチャ変更

【注意事項】

  • 運営の判断で対応を行う場合があります。
  • プレイヤーデータの保存は保証されていません。
  • ルールは予告なく変更されることがあります。
